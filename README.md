# Begley Handyman Service

This is a portfolio website built for my brother's handyman service, **Begley Handyman Service**.

The site showcases his services, process, and commitment to precision home care, allowing clients to easily get in touch and book services.

## Tech Stack

This project is built with:
- **React** for the UI library
- **Vite** for the build tool and development server
- **Tailwind CSS** for styling
- **Bun** as the package manager and runtime

## Getting Started

### Prerequisites

You need to have [Bun](https://bun.sh/) installed on your machine.

### Installation

1. Clone the repository
2. Install dependencies:

```bash
bun install
```

### Development

To start the development server:

```bash
bun run dev
```

### Build

To build the project for production:

```bash
bun run build
```
